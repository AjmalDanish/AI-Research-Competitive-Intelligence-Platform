# Archive Directory

This directory contains functionality that has been preserved but is not part of the MVP scope for the AI Website Intelligence Platform.

## Structure

- `frontend/` - Complete React application with Material-UI dashboard, authentication, and CRM interfaces
- `crm-features/` - Backend modules for competitive intelligence, user management, alerts, and reports
- `documentation/` - Documentation specific to the previous CRM-focused platform
- `database/` - Database artifacts from the SQLite-based implementation

## Purpose

These files are preserved to:
1. Maintain Git history
2. Allow potential reference during development
3. Enable future re-use if business requirements evolve

## Do Not Delete

This directory contains production code that may be useful for future phases or feature additions.

## Migration Path

If any archived functionality needs to be restored in the future:
1. Review the Git history for context
2. Refactor to align with Clean Architecture
3. Integrate into the new modular structure
