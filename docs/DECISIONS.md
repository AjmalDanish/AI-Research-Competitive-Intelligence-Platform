# Engineering Decisions

This document provides a high-level overview of the important engineering decisions made throughout the project.

Detailed Architecture Decision Records (ADRs) are stored in the `docs/adr/` directory.

---

## Decision Process

Every significant architectural decision should answer four questions:

1. What problem are we solving?
2. Which alternatives were considered?
3. Why was this solution selected?
4. What are the trade-offs?
