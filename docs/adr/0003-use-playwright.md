# ADR-0003

# Use Playwright

Status

Accepted

---

## Context

Many websites render content using JavaScript.

Traditional HTTP requests miss dynamic content.

---

## Alternatives

Selenium

Requests

Scrapy

---

## Decision

Use Playwright.

---

## Reason

Supports

- Chromium
- Firefox
- WebKit
- Headless execution
- Modern JavaScript rendering

---

## Consequences

Pros

Reliable rendering

Excellent automation

Modern API

Cons

Higher resource usage
